Preparar el Escenario

Instrucciones

1. Abrir la terminal de comandos:

  • Presiona las teclas #1 para abrir la ventana de "Ejecutar".
  • Escribe cmd en el campo de texto y presiona Enter o haz clic en "Aceptar". Esto abrirá la terminal de comandos de Windows (símbolo del sistema).

2. Navegar a la ubicación deseada:

  • Una vez que tengas la terminal abierta, usa el comando cd (cambiar directorio) para navegar a la carpeta donde deseas crear tu proyecto Django.

Por ejemplo, si deseas ir a la carpeta Documentos:

```{ .bash .docpath title="terminal" } cd ~/Documents/ ```

3. Crea una nueva carpeta para el proyecto:

  • El siguiente comando crea la carpeta llamada y entra en ella:
terminal 
        
        mkdir django_project && cd django_project

4. Crear un entorno virtual:

  • Ejecuta el siguiente comando para crear un entorno virtual en la carpeta del proyecto:
terminal 
        
        python3 -m venv venv

5. Activar entorno virtual:

  • En Windows, activa el entorno virtual con el siguiente comando:
terminal 
        
        source venv/bin/activate

6. Instalar Django en el entorno virtual:

  • Una vez que el entorno virtual está activado, instala Django con el siguiente comando:
```bash title="terminal" pip install django ```

7. Generar un nuevo proyecto de django:

  • Ahora, ejecuta el siguiente comando para crear el proyecto Django dentro de la carpeta principal :
```bash title="terminal" django-admin startproject _site . ```

Nota

El comando anterior evita crear una subcarpeta con el mismo nombre del proyecto. En su lugar, genera una carpeta llamada _site, que alberga todos los archivos relacionados con la configuración del proyecto.
