Preparar el Escenario

Instrucciones:

1. Crea una nueva carpeta y entra en ella
terminal 
        
        mkdir django_project && cd django_project
2. Crear un entorno virtual
terminal 
        
        python3 -m venv venv
3. Activar entorno virtual
terminal 
        
        source venv/bin/activate
4. Instalar Django ```bash title="terminal" pip install django ``` 5. Generar un nuevo proyecto de django ```bash title="terminal" django-admin startproject _site . ```
